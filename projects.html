my projects

Unity Endless Runner

Using Brackeys' Youtube tutorials as well as free 3D assets from the internet, I made a simple endless runner game.

Discord Music Bot

To combat the problem of many Discord music bots getting removed, I decided to try to make my own with the discord.py library, so that my friends and I could continue listening to music. I also learned some basics of OOP and how to use multiple .py files for a single program :O

Real-time Carpark Information from data.gov.sg

Using the requests module in Python, I learnt how to use APIs to fetch data (in this case from data.gov.sg) instead of just using .csv files. This also made me realise how important APIs are, allowing for the use of real-time updating information in a program easily.

Trivia Website with Leaderboard

Using a trivia API, three.js and RestDB, my two friends and I made a trivia game. This was a really interesting project, opening my eyes to 3D web development and storing/retrieving data from a database. Overall, this was a great and fun project and led me to continue using three.js for future projects (like this site! :D)
